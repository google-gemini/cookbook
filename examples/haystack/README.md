# Gemini API Haystack Examples

This is a collection of examples for the Gemini API used with [Haystack](https://haystack.deepset.ai/).

* [Cross-modal Retrieval with Gemini Embedding and Haystack](https://github.com/google-gemini/cookbook/blob/main/examples/haystack/Gemini_Embedding_Haystack_Crossmodal_Retrieval.ipynb): Use Gemini Embedding 2 with Haystack for textual retrieval, RAG, and cross-modal retrieval across text, images, video, and PDFs.

# Gemini for Robotics

This directory contains a set of notebooks designed to help you get started with Gemini for Robotics.

## Notebooks

We recommend going through the notebooks in the following order:

1.  **[Get Started](get_started.ipynb)**: An introduction to the Gemini Robotics-ER 1.5 model, covering basic setup and object detection.
2.  **[Robotics API Features](robotics_api_features.ipynb)**: Learn about specific API features crucial for robotics, such as JSON mode, thinking budgets, and code execution.
3.  **[Spatial Awareness](spatial_awareness.ipynb)**: Dive deeper into spatial reasoning, including bounding boxes and segmentations.
4.  **[Physical Reasoning](physical_reasoning.ipynb)**: Explore how Gemini can reason about physical properties like weight and fragility.
5.  **[Task Orchestration](task_orchestration.ipynb)**: Learn how to use Gemini to orchestrate complex, multi-step tasks.
